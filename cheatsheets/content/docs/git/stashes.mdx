### Temporarily save changes in the working tree

```bash
git stash save "stash message"
```

### List all stashes

```bash
git stash list
```

### Apply changes from a specific stash

```bash
git stash apply <stash>
```

### Remove a specific stash

```bash
git stash drop <stash>
```

### Remove all stashes

```bash
git stash clear
```
