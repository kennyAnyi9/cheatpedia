#### Golang only has the for loop

```go
for i := 0; i < 10; i++ {
    // i
}
```

#### For as a while

<Callout title="The first and third parameters are ommitable"></Callout>

```go
i := 0;

for i < 10 {
    i++
}
```

#### Forever loop

```go

}
```
